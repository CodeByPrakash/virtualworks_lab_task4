# Quote Generator with History

A modern web application that fetches random, inspirational quotes from an external API, displays them using an elegant UI, and saves each fetched quote to a local database to maintain a persistent history.

## Features

- **External API Integration:** Seamlessly connects to a public quotes API (`dummyjson.com/quotes/random`) to fetch new quotes on demand.
- **Database Storage:** Automatically stores every fetched quote into a local SQLite database (`quotes.db`) to preserve history across sessions.
- **Dynamic Frontend:** Utilizes the Fetch API to retrieve new quotes and refresh the history list asynchronously without reloading the page.
- **Premium UI Design:** Built with a design system generated by the Stitch MCP ("Mindful Clarity"), featuring beautiful serif typography, Tailwind CSS styling, and subtle micro-animations.

## Technology Stack

- **Backend:** Python 3, Flask framework.
- **Database:** SQLite.
- **External API:** `requests` library to fetch data.
- **Frontend:** HTML5, Tailwind CSS (via CDN), and Vanilla JavaScript.

## Setup and Installation

1. Ensure Python 3 is installed.
2. Navigate to the `task_4` directory in your terminal.
3. Install the dependencies. *(Note: If `pip install` gives a fatal error on your Windows machine, try using `python -m pip` instead)*:
   ```bash
   python -m pip install -r requirements.txt
   ```

## Running the Application

1. From the `task_4` directory, start the development server:
   ```bash
   python app.py
   ```
2. The server runs on port **5003** by default. Open your web browser and navigate to:
   [http://127.0.0.1:5003](http://127.0.0.1:5003)

## API Endpoints

- `GET /api/quote` - Reaches out to the external API for a new random quote, saves it to the SQLite database, and returns the quote data to the frontend.
- `GET /api/history` - Fetches all previously saved quotes from the database in reverse-chronological order.
